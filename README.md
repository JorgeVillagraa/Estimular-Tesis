# Estimular-Tesis

Proyecto de tesis para el sistema de gestión de Estimular, una organización dedicada al cuidado y desarrollo de niños.

## Descripción

Este proyecto incluye un backend desarrollado en Node.js con Express y un frontend en React utilizando Vite. Permite gestionar niños, turnos, entrevistas, pagos y más funcionalidades relacionadas con la administración de la organización.

## Tecnologías Utilizadas

- **Backend**: Node.js, Express, PostgreSQL
- **Frontend**: React, Vite
- **Base de Datos**: PostgreSQL
- **Autenticación**: JWT
- **Otros**: Email service, scheduling

## Instalación

### Backend
1. Navega a la carpeta `backend`:
   ```
   cd backend
   ```
2. Instala las dependencias:
   ```
   npm install
   ```
3. Configura la base de datos en `src/config/db.js`.
4. Ejecuta el servidor:
   ```
   npm start
   ```

### Frontend
1. Navega a la carpeta `frontend`:
   ```
   cd frontend
   ```
2. Instala las dependencias:
   ```
   npm install
   ```
3. Ejecuta el servidor de desarrollo:
   ```
   npm run dev
   ```

## Uso

- Accede al frontend en `http://localhost:5173` (puerto por defecto de Vite).
- El backend corre en `http://localhost:3000` (ajusta según configuración).

## Scripts Disponibles

En la raíz del proyecto, hay scripts SQL en `scripts/` para inicializar la base de datos.
